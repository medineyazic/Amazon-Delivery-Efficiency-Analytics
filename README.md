Conducted a comprehensive analysis on 43,000+ Amazon delivery records to identify key factors influencing last-mile delivery efficiency. Applied data preprocessing (MICE imputation, feature engineering, clustering) and built multiple machine learning models, including Multiple Linear Regression, ANN, SVM, Random Forest, and XGBoost.
Found that traffic, weather, delivery area, and courier characteristics significantly impact delivery time.
Achieved the best predictive accuracy with XGBoost (R² = 0.919, RMSE = 14.8), outperforming other models.
Results provided both academic insights and practical implications for improving logistics operations and customer satisfaction.
